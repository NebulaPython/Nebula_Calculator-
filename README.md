# 🚀 Nebula Calculator

**Nebula Calculator** is a powerful, "Swiss Army Knife" desktop scientific calculator built with Python. It combines a sleek, modern UI with a nostalgic **Vintage LCD display** aesthetic, offering a robust set of features for students, engineers, and power users.

Built using the **CustomTkinter** library and compiled, Nebula delivers a native, high-performance experience with full support for System Dark/Light modes.

## ✨ Key Features

* **🧪 Advanced Scientific Suite:** Includes trigonometry (`sin`, `cos`, `tan`), logarithms (`log`, `ln`), factorials (`n!`), absolute values (`abs`), and more.
* **📐 Quadratic Equation Solver:** A dedicated `Eq` module to solve ax² + bx + c = 0 instantly, providing both real and complex root analysis.
* **🧠 Smart Memory System:** Professional-grade memory management with `MC`, `MR`, and `M+` functions to handle multi-step calculations.
* **⌨️ Numpad & Keyboard Integration:** Fully mapped for physical keyboards. Use your Numpad for numbers and operators (+, -, *, /) and Enter for instant results.
* **📺 Vintage LCD Interface:** A custom-designed, retro-themed display area with a monospaced digital font for maximum legibility.
* **🛡️ Secure & Optimized:** Uses a restricted evaluation logic for safe mathematical parsing and is compiled into a standalone executable for speed and security.

## 🛠️ Tech Stack

* **Language:** Python 3.10+
* **GUI:** CustomTkinter
* **Compilation:** Compiled (C++ Translated Executable)
* **Logic:** Math & Safe-Eval Parser
